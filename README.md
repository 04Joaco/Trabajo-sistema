# 🚗 Sistema de Turnos para VTV

## 📌 Descripción

Este proyecto implementa un sistema de gestión de turnos para la Verificación Técnica Vehicular (VTV).  
Permite a los usuarios reservar turnos de manera simple, optimizando la organización y reduciendo tiempos de espera.

---

## 🎯 Objetivos

- Facilitar la reserva de turnos online  
- Evitar la saturación de plantas VTV  
- Mejorar la experiencia del usuario  
- Optimizar la gestión de disponibilidad y horarios  

---

## 🧩 Tecnologías utilizadas

- HTML
- CSS
- JavaScript
